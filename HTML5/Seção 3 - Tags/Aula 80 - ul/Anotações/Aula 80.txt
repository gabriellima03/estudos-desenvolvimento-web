Tag utilizada para definir uma lista não ordenada. Para cada ítem de uma lista dentro dela, devemos usar uma tag "li". Recebe todos os atributos globais e não possui nenhum atributo exclusivo.

Estilo CSS padrão:

ul {
    display: block;
    list-style-type: disc;
    margin-top: 1em;
    margin-bottom: 1em;
    margin-left: 0;
    margin-right: 0;
    padding-left: 40px;
}
