Tag utilizada para definir uma lista com descrição e é utilizada juntamente às tags "dt" e "dd". Recebe todos os atributos globais e não possui nenhum atributo exclusivo.

Estilo CSS padrão:

dl {
    display: block;
    margin-top: 1em;
    margin-bottom: 1em;
    margin-left: 0;
    margin-right: 0;
}
