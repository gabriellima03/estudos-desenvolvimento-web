Essa tag é utilizada para agrupar texto ou tags em um documento e não fornece nenhuma alteração visual por padrão. Geralmente é utilizada para aplicar algum estilo CSS. Ela recebe todos os atributos globais e não possui nenhum atributo exclusivo. Ela não possui nenhum estilo CSS padrão. Ela geralmente é utilizada para estilizar algum texto dentro de um elemento sem causar alteração no peso semântico.
