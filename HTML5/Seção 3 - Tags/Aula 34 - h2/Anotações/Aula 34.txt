Essa tag é utilizada para definir o segundo cabeçalho mais importante da página. Ela recebe todos os atributos globais e não possui nenhum atributo exclusivo. Diferentemente da tag "h1" o uso múltiplo de "h2" não é desencorajado, por se tratar de subtítulo.

Estilo CSS padrão:

h2 {
    display: block;
    font-size: 1.5em;
    margin-top: 0.83em;
    margin-bottom: 0.83em;
    margin-left: 0;
    margin-right: 0;
    font-weight: bold;
}
