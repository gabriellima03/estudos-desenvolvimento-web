Essa tag é utilizada para definir o quarto cabeçalho mais importante da página. Ela recebe todos os atributos globais e não possui atributos exclusivos.

Estilo CSS padrão:

h4 {
    display: block;
    font-size: 1em;
    margin-top: 1.33em;
    margin-bottom: 1.33em;
    margin-left: 0;
    margin-right: 0;
    font-weight: bold;
}
