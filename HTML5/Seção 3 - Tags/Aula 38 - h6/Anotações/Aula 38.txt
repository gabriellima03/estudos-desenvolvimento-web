Tag utilizada para definir o sexto e último cabeçalho mais importante da página. Ela possui todos os atributos globais e não tem nenhum atributo exclusivo.

Estilo CSS padrão:

h6 {
    display: block;
    font-size: .67em;
    margin-top: 2.33em;
    margin-bottom: 2.33em;
    margin-left: 0;
    margin-right: 0;
    font-weight: bold;
}
